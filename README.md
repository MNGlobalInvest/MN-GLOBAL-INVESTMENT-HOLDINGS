## Overview

**MN Global Investment Holdings LLC** operates at the intersection of artificial intelligence and Bitcoin protocol development. We provide enterprise-grade AI services while maintaining fully segregated, deterministic Bitcoin accumulation through protocol-level automation.

### What We Do

- **AI-Driven Revenue Services** - Production-ready AI modules generating USD revenue
- **Protocol-Enforced Bitcoin Accumulation** - Deterministic BTC flows via Taproot scripting  
- **Self-Healing Infrastructure** - Autonomous operational capabilities with 24/7 monitoring
- **Real-World Deployment** - Built with existing cloud services, APIs, and open-source tools

## Core Architecture

### Bitcoin Protocol Layer

```yaml
Node Infrastructure: Bitcoin Core with Taproot script support
Operating System: Ubuntu 22.04 LTS
Security Model: Cold storage + 2-of-3 multisig
Hardware Wallets: Ledger, Trezor integration
Verification: Independent node validation with automated audit trails
Deployment: AWS/GCP free tier or on-premise servers
```

**Key Features:**
- Deterministic BTC accumulation via Taproot scripts
- Zero online key storage - all keys in cold/hardware wallets
- Multi-node verification across geographic locations
- Automated compliance and audit logging

### Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/MNGlobal/holdings.git
cd holdings

# 2. Environment setup
cp .env.example .env
# Edit .env and configure your API keys

# 3. Deploy Bitcoin node (testnet first)
./scripts/deploy-node.sh --network testnet

# 4. Deploy AI modules
docker-compose up -d

# 5. Verify deployment
./scripts/health-check.sh
```
